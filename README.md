# 韭研公社文章管理器

## 快速开始

### 1. 首次登录
```bash
python src/login_save_cookies.py
```

### 2. 启动程序
```bash
python src/article_manager_gui.py
```

## 目录结构

```
JC/
├── src/           # 源代码
├── data/          # 数据文件
├── logs/          # 日志
├── config/        # 配置
├── tools/         # 工具
├── docs/          # 文档
└── temp/          # 临时文件
```

## 功能特性

- ⏰ 定时采集（每 30 分钟）
- 💾 增量存储（自动去重）
- 🔍 搜索筛选
- 📤 导出分享
- 📊 数据统计
