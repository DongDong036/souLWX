### AI 系统分析

- **UI 组件库**：Element UI
- **渲染方式**：服务端渲染 (SSR)
- **数据加载**：JavaScript 动态渲染

- 支持排序方式：最新热度、最新发布、最新互动、30天热度

**文章详情页** (`https://www.jiuyangongshe.com/a/{article_id}`)
