### 二、新增功能模块

#### 1. 日志模块 (`src/logger.py`)
- ✅ 统一的日志管理
- ✅ 支持日志轮转（按天/按大小）
- ✅ 分级记录（INFO/WARNING/ERROR）
- ✅ 自动清理旧日志（可配置）

**使用示例**：
```python
from logger import log_app, log_collect, log_error

log_app('应用程序启动')
log_collect('开始采集任务')
log_error('发生错误', level='error')
```

#### 2. 配置管理模块 (`src/config_manager.py`)
- ✅ 统一的配置加载和保存
- ✅ 路径管理函数
- ✅ 默认配置支持
- ✅ 配置验证

**使用示例**：
```python
from config_manager import load_config, get_database_path

config = load_config()
db_path = get_database_path()
```
